<h1>Inventory Management Dashboard</h1>
<p>A web-based inventory management system built using Django + Tailwind CSS.
This project allows businesses to manage products, suppliers, and stock efficiently.</p>

Features
- User Authentication (Login/Logout)
- Dashboard with key statistics
- Product Management (Add, Edit, Delete with Image Upload)
- Supplier Management (Add, Edit, Delete)
- Clean UI using Tailwind CSS
- Popup Modals for better user experience

Tech Stack
- Backend: Django (Python)
- Frontend: HTML, Tailwind CSS, JavaScript
- Database: SQLite (default Django DB)
